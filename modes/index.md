# 批改模式

AI 批改助手提供三种批改模式，适用于不同的使用场景。选择合适的模式可以大幅提升批改效率。

## 模式对比

| | 普通模式 | 试改模式 | 无人值守模式 |
|--|---------|---------|------------|
| 适合场景 | 日常少量批改 | 首次使用、调优评分标准 | 大量试卷、夜间挂机 |
| 提交速度 | 5 秒倒计时（可暂停/取消） | 等待教师确认 | 1 秒自动提交 |
| 错误处理 | 弹窗提示 | 弹窗提示 | 自动重试，最多 3 次 |
| 纠错支持 | ✅ | ✅ | ❌ |

::: tip 如何选择？
- **第一次使用**：先用试改模式，确认 AI 评分准确后再切换
- **日常少量批改**：普通模式，每份试卷都有确认机会
- **大批量 / 夜间挂机**：无人值守模式，睡一觉起来全改完
:::
