# 批改模式

AI 批改助手提供多种批改模式，适用于不同的使用场景。选择合适的模式可以大幅提升批改效率。

::: info 移动端用户
如果你在手机上浏览，可以点击页面左上角的 **☰ 菜单按钮** 展开侧边目录，在各章节之间切换。
:::

## 模式对比

| | 普通模式 | 试改模式 | 无人值守模式 | 双评模式 |
|--|---------|---------|------------|---------|
| 适合场景 | 日常少量批改 | 首次使用、调优评分标准 | 大量试卷、夜间挂机 | 高精度要求 |
| 提交速度 | 5 秒倒计时（可暂停/取消） | 等待教师确认 | 1 秒自动提交 | 取决于所选模式 |
| 错误处理 | 弹窗提示 | 弹窗提示 | 自动重试，最多 3 次 | 弹窗提示 |
| 纠错支持 | 支持 | 支持 | 不支持 | 支持 |
| 评分方式 | 单模型 | 单模型 | 单模型 | 双模型 + 仲裁 |

## 快速导航

1. **[普通模式](/modes/normal)** — 日常批改模式，5 秒倒计时提交
2. **[试改模式](/modes/trial)** — 首次使用推荐，等待确认后再提交
3. **[无人值守模式](/modes/unattended)** — 挂机批改模式，1 秒自动提交
4. **[双评模式](/modes/dual)** — 高精度模式，两个模型独立评分，超阈值自动仲裁

::: tip 如何选择？
- **第一次使用**：先用试改模式，确认 AI 评分准确后再切换
- **日常少量批改**：普通模式，每份试卷都有确认机会
- **大批量 / 夜间挂机**：无人值守模式，睡一觉起来全改完
- **高精度要求**：双评模式，两个模型交叉验证，适合重要考试
:::
