# 项目介绍

**AI 批改助手** (AI-Marker-Suite) 是一个 Tampermonkey（油猴）用户脚本，能够在在线阅卷平台上自动批改主观题试卷。

> **晚上挂机睡觉，早上起来全改完。**

## 它能做什么？

AI 批改助手通过多模态 AI 视觉识别技术，自动完成以下流程：

1. **抓取答题卡图片** — 从阅卷页面自动获取学生答题卡图片
2. **AI 识别与评分** — 将图片发送给 AI，识别手写答案并根据评分标准打分
3. **自动填分提交** — 将分数自动填入输入框并提交，切换到下一份试卷

整个过程无需人工干预，支持连续自动批改。

## 核心特性

### 智能评分

- AI 自动识别手写答案，根据你设定的评分标准打分
- 生成评语和得分理由，可实时查看 AI 思考过程
- 支持**分小题评分**：一道大题拆成多个小题分别打分（如作文分内容/语言/结构）

### 三种批改模式

|  | 普通模式 | 试改模式 | 无人值守模式 |
|--|---------|---------|------------|
| 适合场景 | 日常少量批改 | 首次使用、调优评分标准 | 大量试卷、夜间挂机 |
| 提交速度 | 5 秒倒计时（可暂停/取消） | 等待教师确认 | 1 秒自动提交 |
| 错误处理 | 弹窗提示 | 弹窗提示 | 自动重试，最多 3 次 |
| 纠错支持 | 支持 | 支持 | 不支持 |

### 多平台支持

通过适配器模式同时支持多个阅卷平台：

- **智学网** (zhixue.com)
- **七天网络** (7net.cc / qt7.net) — 含旧版和新 UI

### 多方案管理

- 不同题目（语文作文 / 数学大题 / 英语翻译）保存不同配置
- 绑定题目 URL，打开自动切换，不用每次手动选

### 分数纠错与学习

- AI 打分不准？输入正确分数，AI 自动分析原因并优化评分标准
- 越用越准，AI 会从教师反馈中持续学习

### 评阅历史

- 独立浮动按钮，随时查看批改记录
- 支持导出 HTML 报告（含答题卡图片），方便存档和复查

### 多服务商兼容

- 内置 5+1 AI 推荐服务（免费注册）
- 也支持 DeepSeek、火山引擎、硅基流动等任何 OpenAI 兼容接口

## 技术架构

项目采用**模块化 + 适配器模式**架构：

- `src/core/` — 与平台无关的核心逻辑（AI 调用、UI 组件、状态管理、历史记录等）
- `src/adapters/` — 平台特定的 DOM 交互逻辑（智学网、七天网络等）
- 构建时所有模块合并为单个 `.user.js` 文件
- 运行时自动检测当前平台并加载对应适配器
