# 常见问题

## 基本使用

### 识别不准怎么办？

填写「题目内容」「标准答案」「评分标准」三项，准确率会大幅提升。AI 需要这些上下文信息才能做出准确判断。

建议先用**试改模式**批改几份试卷，观察 AI 的识别和评分结果，根据实际情况调整评分标准。

### 遇到 403 错误怎么办？

脚本会自动检测 403 错误并刷新页面，无需手动处理。这通常是阅卷平台的会话过期导致的。

### 无人值守模式安全吗？

建议先用普通模式批改几份试卷，确认 AI 评分准确后再开启无人值守模式。无人值守模式下：

- 提交间隔为 1 秒
- AI 识别失败时自动重试，最多 3 次
- 连续失败会自动停止，避免误操作

### 支持哪些题型？

所有有手写答案的**主观题**都支持，包括：

- 作文
- 解答题
- 论述题
- 简答题
- 翻译题

客观题（选择题、填空题等）请使用阅卷平台自带的功能。

### 怎么停止批改？

点击右下角的浮动按钮暂停，在弹窗中点击「取消并退出」即可停止。

### 弹窗没出现？

按 `F12` 打开浏览器控制台，查看带 `[诊断]` 标记的日志，截图反馈给作者。

## 配置相关

### API 密钥在哪里获取？

推荐使用 5+1 AI 服务：[获取 API KEY](https://api.ai.five-plus-one.com/console/token)（免费注册）。

也支持任何 OpenAI 兼容接口，在设置面板中填写对应的端点地址、密钥和模型名即可。

### 支持哪些 AI 服务商？

任何兼容 OpenAI `/v1/chat/completions` 接口的服务商都支持，包括但不限于：

- **5+1 AI**（推荐，内置默认配置）
- DeepSeek
- 火山引擎
- 硅基流动
- OpenAI

### 怎么为不同题目保存不同配置？

使用**预设方案管理**功能。详见 [预设方案管理](../config/presets)。

简单来说：

1. 创建新方案，填写题目内容、标准答案、评分标准
2. 将方案绑定到对应题目的 URL
3. 之后打开该题目时会自动切换到对应方案

### 配置会丢失吗？

配置保存在浏览器的 Tampermonkey 存储中，除非手动清除浏览器数据，否则不会丢失。

脚本还支持**配置导出/导入**功能，可以将所有配置导出为 JSON 文件备份。

## 批改相关

### 分数填入失败？

可能是阅卷平台页面结构发生了变化。请检查：

1. 当前页面是否为批改页面（包含答题卡和分数输入框）
2. 脚本是否为最新版本
3. 控制台是否有报错信息

### AI 一直返回 "未能识别"？

可能的原因：

1. 图片质量太低或图片加载失败
2. AI 服务暂时不可用
3. API 密钥额度用尽

建议在设置面板中切换到「流式输出」模式，查看 AI 的实时响应。

### 纠错功能怎么使用？

在**普通模式**或**试改模式**下，AI 打分后会弹出提交确认对话框。如果分数不准确：

1. 点击对话框中的纠错按钮
2. 输入正确的分数和评分理由
3. AI 会自动分析差异原因，并优化评分标准

详见 [分数纠错](../advanced/correction)。

### 怎么查看批改历史？

点击页面右下角的**历史按钮**（时钟图标），可以查看所有批改记录，支持：

- 按时间、方案筛选
- 查看详情（含答题卡图片）
- 导出为 CSV / JSON / HTML 报告
- 标记不正确的评分以便回评

## 平台相关

### 智学网和七天网络有什么区别？

脚本对两个平台的核心功能一致，差异在于页面结构和交互方式。脚本会自动检测当前平台并加载对应的适配器，用户无需手动选择。

### 支持其他阅卷平台吗？

目前支持智学网和七天网络。如果你需要适配其他平台，请 [联系作者](https://r-l.ink/contact) 或参考 [开发新适配器](../advanced/adapter-dev) 自行开发。
