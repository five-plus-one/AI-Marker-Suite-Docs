# 常见问题

## 安装与环境

### 手机能用吗？

**不能。** AI 批改助手是一个浏览器脚本，必须在电脑端的 Edge 浏览器中运行。手机、平板、微信内置浏览器均不支持。

### 用什么浏览器？

推荐使用 **Edge 浏览器**（Windows 10/11 自带）。Chrome 也可以，但其扩展商店在国内无法直接访问。

以下浏览器**不支持**：360 安全浏览器、搜狗浏览器、QQ 浏览器等国产浏览器。

### 什么是 Tampermonkey（油猴）？

Tampermonkey 是一个免费的浏览器扩展（插件），它可以让浏览器运行额外的脚本程序。AI 批改助手就是通过它来运行的。你只需要安装一次，之后脚本会自动更新。

### 稳定版、预览版、开发版有什么区别？

AI 批改助手提供三个版本渠道：

| 渠道 | 说明 | 安装方式 |
|------|------|---------|
| **稳定版** | 经过充分测试，推荐日常使用 | [一键安装](https://auto-update.aimarking.five-plus-one.com/ota/ai_marker.user.js) |
| **预览版** | 包含最新功能，可能不够稳定 | [一键安装](https://auto-update.aimarking.five-plus-one.com/ota/preview/ai_marker.user.js) |
| **开发版** | 面向开发者的自动构建版本 | 不提供公开安装链接 |

安装后可以在脚本设置中随时切换渠道。如遇到问题，建议切换回稳定版。

### 安装后看不到浮动按钮？

脚本只在阅卷页面（能看到答题卡图片和分数输入框的页面）上才会激活。请确认：

1. 当前页面是智学网、七天网络、好分数、五岳阅卷、华翰云、光大阅卷、云阅卷、新教育、润建学情或54学霸的阅卷页面
2. 页面上能看到学生答题卡图片
3. Tampermonkey 扩展已启用（浏览器右上角图标为黑色，不是灰色）

## 基本使用

### 识别不准怎么办？

填写「题目内容」「标准答案」「评分标准」三项，准确率会大幅提升。AI 需要这些上下文信息才能做出准确判断。

建议先用**试改模式**批改几份试卷，观察 AI 的识别和评分结果，根据实际情况调整评分标准。

### 遇到 403 错误怎么办？

脚本会自动检测 403 错误并刷新页面，无需手动处理。这通常是阅卷平台的会话过期导致的。

### 无人值守模式安全吗？

建议先用普通模式批改几份试卷，确认 AI 评分准确后再开启无人值守模式。无人值守模式下：

- 提交间隔为 1 秒
- AI 识别失败时自动重试，最多 3 次
- 连续失败会自动停止，避免误操作

### 支持哪些题型？

所有有手写答案的**主观题**都支持，包括：

- 作文
- 解答题
- 论述题
- 简答题
- 翻译题

客观题（选择题、填空题等）请使用阅卷平台自带的功能。

### 怎么停止批改？

点击右下角的浮动按钮暂停，在弹窗中点击「取消并退出」即可停止。

### 弹窗没出现？

按 `F12` 打开浏览器控制台，查看带 `[诊断]` 标记的日志，截图反馈给作者。

## 配置相关

### API 密钥在哪里获取？

推荐使用 5+1 AI 服务：[获取 API KEY](https://api.ai.five-plus-one.com/console/token)。新注册用户有少量试用额度，用完后需充值。

也支持火山引擎、硅基流动等其他 OpenAI 兼容接口，但所选模型**必须支持图片输入**。

### 支持哪些 AI 服务商？

任何兼容 OpenAI `/v1/chat/completions` 接口且**支持图片输入（多模态）**的服务商都支持：

- **5+1 AI**（推荐，内置默认配置）
- 火山引擎（豆包）
- 硅基流动

::: warning DeepSeek 不能用
DeepSeek 的模型是纯文本模型，不支持图片输入，无法识别手写答案。请不要选择 DeepSeek。
:::

### 怎么为不同题目保存不同配置？

使用**预设方案管理**功能。详见 [预设方案管理](../config/presets)。

简单来说：

1. 创建新方案，填写题目内容、标准答案、评分标准
2. 将方案绑定到对应题目的 URL
3. 之后打开该题目时会自动切换到对应方案

### 配置会丢失吗？

配置保存在浏览器的 Tampermonkey 存储中，除非手动清除浏览器数据，否则不会丢失。

脚本还支持**配置导出/导入**功能，可以将所有配置导出为 JSON 文件备份。

## 批改相关

### 分数填入失败？

可能是阅卷平台页面结构发生了变化。请检查：

1. 当前页面是否为批改页面（包含答题卡和分数输入框）
2. 脚本是否为最新版本
3. 控制台是否有报错信息

### AI 一直返回 "未能识别"？

可能的原因：

1. 图片质量太低或图片加载失败
2. AI 服务暂时不可用
3. API 密钥额度用尽

建议在设置面板中切换到「流式输出」模式，查看 AI 的实时响应。

### 纠错功能怎么使用？

在**普通模式**或**试改模式**下，AI 打分后会弹出提交确认对话框。如果分数不准确：

1. 点击对话框中的纠错按钮
2. 输入正确的分数和评分理由
3. AI 会自动分析差异原因，并优化评分标准

详见 [分数纠错](../advanced/correction)。

### 怎么查看批改历史？

点击页面右下角的**历史按钮**（时钟图标），可以查看所有批改记录，支持：

- 按时间、方案筛选
- 查看详情（含答题卡图片）
- 导出为 CSV / JSON / HTML 报告
- 标记不正确的评分以便回评

## 平台相关

### 支持哪些阅卷平台？

目前支持以下十个平台，脚本会自动检测并加载对应适配器：

- **智学网** (zhixue.com) — 最早支持的平台
- **七天网络** (7net.cc / qt7.net / yj5.7net.cc) — 支持新旧两种 UI
- **好分数** (haofenshu.com) — v1.11.3 新增支持
- **五岳阅卷** (wylkyj.com) — v1.12.1 新增支持
- **华翰云** (yunyuejuan.net) — v1.21.2 新增支持
- **光大阅卷** (pj.yixx.cn) — v1.21.2 新增支持
- **云阅卷** — 内网阅卷系统
- **新教育** (xinjiaoyu.com) — 支持考试和作业两种批改模式
- **润建学情** (aisusheng.runjian.com) — 支持 CSS 背景图和点击式评分
- **54学霸** (54xueba.cn) — 支持 Canvas/base64 图片和多小题输入框

### 不同平台有什么区别？

脚本对所有平台的核心功能一致（识别 → 打分 → 填入 → 提交），差异在于页面结构和交互方式。脚本会自动检测当前平台并加载对应的适配器，用户无需手动选择。

### 支持其他阅卷平台吗？

目前支持智学网、七天网络、好分数、五岳阅卷、华翰云、光大阅卷、云阅卷、新教育、润建学情和54学霸。如果你需要适配其他平台，请 [联系作者](https://r-l.ink/contact) 或参考 [开发新适配器](../advanced/adapter-dev) 自行开发。
