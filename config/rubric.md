# 评分规则

评分规则是 AI 批改助手的核心配置，直接影响 AI 打分的准确度。配置越详细，AI 评分越精准。

## 三个关键字段

在设置面板中，你需要填写以下三个字段：

### 题目内容

试卷上的**原始题目**，告诉 AI 这道题在问什么。

**示例：**
```
阅读下面的材料，根据要求写作。
"路虽远，行则将至；事虽难，做则必成。"
请以"坚持"为话题，写一篇不少于800字的议论文。
```

### 标准答案

**参考答案**或**评分样例**，告诉 AI 什么样的答案是好的。

**示例：**
```
优秀的议论文应当：
1. 有明确的中心论点（坚持的重要性）
2. 使用至少2个事实论据和1个道理论据
3. 论证结构清晰：引论-本论-结论
4. 语言流畅，无明显语病
5. 字数达到800字以上
```

### 评分标准

**给分/扣分细则**，告诉 AI 如何分配分数。这是最重要的字段。

**示例：**
```
满分40分，分项评分：
- 立意(10分)：观点明确、切合题意得8-10分；基本切题得5-7分；偏离题意得0-4分
- 内容(10分)：论据充实、论证有力得8-10分；论据单薄得5-7分；内容空洞得0-4分
- 结构(10分)：结构完整、层次清晰得8-10分；结构基本完整得5-7分；结构混乱得0-4分
- 语言(10分)：语言流畅、有文采得8-10分；语言通顺得5-7分；语病较多得0-4分
- 每少50字扣1分，最多扣3分
```

## 编写技巧

### 具体化评分标准

| 不好的写法 | 好的写法 |
|-----------|---------|
| "根据质量给分" | "观点明确(5分)，论据充分(5分)，结构完整(5分)，语言流畅(5分)" |
| "答案正确得满分" | "列式正确得3分，计算过程正确得4分，最终答案正确得3分" |
| "酌情给分" | "指出一种修辞手法得2分，分析效果得1分，满分6分" |

### 提供分数梯度

为每个评分维度提供不同分数档的描述，帮助 AI 更准确地定位分数：

```
内容评分(20分)：
- 18-20分：内容丰富，紧扣主题，有独到见解
- 14-17分：内容较充实，基本切合题意
- 10-13分：内容一般，部分偏离题意
- 6-9分：内容单薄，偏离题意较多
- 0-5分：内容空洞或完全偏题
```

### 包含扣分项

明确列出扣分规则，避免 AI 遗漏：

```
扣分项：
- 错别字：每个扣0.5分，最多扣3分
- 标点错误：每3处扣0.5分，最多扣2分
- 字数不足：每少50字扣1分
- 未写标题：扣2分
```

## 不同题型的配置建议

### 作文类

- 重点描述评分维度（立意、内容、结构、语言）
- 提供不同分数档的详细描述
- 列出扣分项（错别字、字数不足等）

### 解答题

- 明确每个步骤的分值
- 包含常见的解题方法
- 说明计算过程和最终答案的分值分配

### 论述题

- 列出需要涵盖的要点
- 说明每个要点的分值
- 描述不同深度的答案对应的分数

### 翻译题

- 提供标准译文
- 列出关键词的翻译分值
- 说明语法正确性和流畅性的权重

## 纠错优化

如果 AI 打分不准确，可以使用**分数纠错**功能：

1. 输入正确的分数和理由
2. AI 会自动分析差异原因
3. 自动生成优化后的参考答案和评分标准
4. 确认后自动更新配置

经过几轮纠错后，评分会标准会越来越准确。详见 [分数纠错](../advanced/correction)。
