# 配置指南

AI 批改助手提供了灵活的配置选项，让你可以根据不同科目和题目自定义批改行为。

## 文档导航

<CardGrid>
<Card title="AI 接口配置" href="/config/api-setup" description="配置 AI 服务商的 API 密钥，选择适合的多模态 AI 模型" />
<Card title="评分规则" href="/config/rubric" description="自定义评分标准，让 AI 按照你的要求打分" />
<Card title="预设方案管理" href="/config/presets" description="为不同题目保存不同的配置方案，支持 URL 绑定自动切换" />

</CardGrid>

::: tip 建议配置顺序
1. **AI 接口配置** → 首先配置 AI 服务（必须）
2. **评分规则** → 设定评分标准
3. **分小题配置** → 如需分小题评分，按需配置
4. **预设方案管理** → 保存配置以便复用
:::
