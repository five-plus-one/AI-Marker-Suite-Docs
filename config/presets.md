# 预设方案管理

预设方案允许你为不同题目保存不同的配置（题目内容、标准答案、评分标准、AI 服务商等），并支持自动切换。

## 什么是预设方案？

每个预设方案包含一组完整的配置：

- **题目内容** — 试卷上的原始题目
- **标准答案** — 参考答案
- **评分标准** — 给分/扣分细则
- **AI 服务商** — 使用哪个 AI 服务
- **批改模式** — 普通/试改/无人值守
- **分小题配置** — 小题列表和满分

## 基本操作

### 创建新方案

1. 打开设置面板
2. 在「预设方案」区域点击下拉菜单
3. 选择「新建方案」
4. 输入方案名称（如 "语文作文"、"数学大题"）
5. 填写题目内容、标准答案、评分标准等配置
6. 点击保存

### 切换方案

在设置面板顶部的预设方案下拉菜单中选择即可。切换后所有配置会立即更新为该方案的设置。

### 删除方案

在预设方案下拉菜单中选择要删除的方案，点击删除按钮。

::: warning 注意
删除方案后无法恢复，请确认后再操作。「默认配置」方案无法删除。
:::

## URL 绑定（自动切换）

预设方案支持绑定到特定的 URL。绑定后，当你打开对应的阅卷页面时，脚本会自动切换到匹配的方案。

### 设置绑定

1. 打开目标题目的阅卷页面
2. 选择或创建对应的预设方案
3. 在设置面板中点击「绑定当前 URL」
4. 确认绑定

### 工作原理

脚本会监控 URL 变化（包括路径和 hash），当检测到新题目时：

1. 检查是否有方案绑定了当前 URL
2. 如果有，自动切换到该方案
3. 如果没有，恢复为「默认配置」
4. 弹出 Toast 通知告知切换结果

### 解绑

在设置面板中点击「解除绑定」即可取消当前 URL 与方案的绑定关系。

## 使用场景示例

### 场景一：语文老师批改作文

创建两个方案：

| 方案 | 题目内容 | 评分标准 |
|------|---------|---------|
| 大作文 | 800字议论文 | 立意(10)+内容(10)+结构(10)+语言(10) |
| 微写作 | 200字应用文 | 内容(5)+格式(3)+语言(2) |

分别绑定到对应的 URL，打开不同题目时自动切换。

### 场景二：数学老师批改解答题

| 方案 | 题目 | 评分标准 |
|------|------|---------|
| 函数题 | 求函数单调性 | 列式(3)+求导(3)+讨论(4)+结论(2) |
| 几何题 | 证明线面平行 | 作辅助线(2)+证明(6)+结论(2) |

### 场景三：使用不同 AI 服务商

| 方案 | AI 服务商 | 适用场景 |
|------|----------|---------|
| 作文 | 5+1 AI (mimo-v2.5) | 作文需要较强的中文理解能力 |
| 数学 | DeepSeek | 数学推理可能在某些模型上更准确 |

## 配置导入/导出

脚本支持将所有配置导出为 JSON 文件，方便备份和迁移。

### 导出配置

1. 打开设置面板
2. 点击「导出配置」按钮
3. 浏览器会下载一个 JSON 文件，包含所有预设方案和设置

### 导入配置

1. 打开设置面板
2. 点击「导入配置」按钮
3. 选择之前导出的 JSON 文件
4. 确认导入（会覆盖当前配置）

::: tip 备份建议
在更换浏览器或电脑前，建议先导出配置文件。导出的 JSON 文件不包含 API 密钥，导入后需要重新填写。
:::
