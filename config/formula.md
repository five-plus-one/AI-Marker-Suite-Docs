# 公式语法指南

AI 批改助手支持在**题目内容**、**标准答案**和**评分标准**中使用数学公式。只需用简单的语法标记，系统会自动将其渲染为美观的数学排版。

::: tip 不会写公式？
别担心！编辑器提供了三种方式帮你快速搞定：

1. **🪄 AI 润色**：用自然语言描述（如"2倍根号下g"），自动生成公式
2. **✨ AI 识别**：拍照或截图公式，自动转为 LaTeX
3. **fx 模板面板**：点击常用公式模板，直接插入

详见下方 [AI 辅助功能](#ai-辅助功能) 章节。
:::

## 快速上手

### 两种公式格式

**行内公式** — 在文字中嵌入，用单个 `$` 包裹：

```
已知 $a = 3$，$b = 4$，则 $a + b = 7$
```

渲染效果：已知 $a = 3$，$b = 4$，则 $a + b = 7$

**块级公式** — 独立成行、居中显示，用双 `$$` 包裹：

```
$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
```

渲染效果：

$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

::: info 什么时候用哪种？
- **行内公式**：正文中提到的变量、简单表达式，如 $x + 1$、$a^2$
- **块级公式**：重要的结论公式、复杂的推导结果，如求根公式、积分公式
:::

### 打开编辑器

在脚本的设置面板中，点击**题目内容**、**标准答案**或**评分标准**旁边的「编辑」按钮，即可打开全屏 Markdown 编辑器。

| 按钮 | 功能 | 说明 |

| **fx** | 公式模板 | 打开常用公式模板面板，点击即可插入 |

编辑器左侧是代码输入区，右侧是实时预览区。输入公式后，预览区会自动显示渲染效果。

## AI 辅助功能

如果你不熟悉 LaTeX 语法，编辑器提供了三种 AI 辅助方式。

### AI 润色（🪄 按钮）

用**自然语言描述**公式内容，AI 会自动转换为标准 LaTeX 格式。

**操作步骤：**

1. 在编辑器中选中一段自然语言文字（或不选中任何内容）
2. 点击工具栏的 **🪄 AI润色** 按钮
3. 如果没有选中文字，会弹出输入框，输入公式描述即可
4. AI 自动将描述转换为 LaTeX，替换原有内容

**示例：**

| 你输入的描述 | AI 转换结果 |
|:---|:---|
| 2倍根号下g乘d | $2\sqrt{gd}$ |
| x的平方加y的平方等于r的平方 | $x^{2} + y^{2} = r^{2}$ |
| 从1到n的求和a下标i | $\sum_{i=1}^{n} a_{i}$ |

### AI 识别（✨ 按钮）

从**图片中识别**数学公式并转换为 LaTeX。

**操作步骤：**

1. 点击工具栏的 **✨ AI识别** 按钮
2. 选择包含公式的图片（可以是截图、拍照等）
3. AI 自动识别图片中的文字和公式
4. 预览确认后，点击「确认插入」

::: info 前置条件
AI 识别需要先配置 AI 供应商和 API Key。详见 [AI 接口配置](./api-setup)。
:::

## 基础语法速查

### 上标与下标

上标用 `^`，下标用 `_`。**多个字符必须用花括号 `{}` 包裹**。

| 含义 | 语法 | 效果 |
|:---|:---|:---|
| 平方 | `x^{2}` | $x^{2}$ |
| 下标 | `x_{i}` | $x_{i}$ |
| 上下标 | `x_{i}^{2}` | $x_{i}^{2}$ |
| 复杂上下标 | `a_{n+1}^{k-1}` | $a_{n+1}^{k-1}$ |

::: warning 注意
`x^2` 可以工作（单个字符），但 `x^n+1` 会渲染错误（只会上标 n），必须写成 `x^{n+1}`。
:::

### 分数

用 `\frac{分子}{分母}` 表示分数。

| 含义 | 语法 | 效果 |
|:---|:---|:---|
| 简单分数 | `\frac{a}{b}` | $\frac{a}{b}$ |
| 复杂分数 | `\frac{x+1}{x-1}` | $\frac{x+1}{x-1}$ |
| 嵌套分数 | `\frac{1}{1+\frac{1}{x}}` | $\frac{1}{1+\frac{1}{x}}$ |

### 根号

用 `\sqrt{}` 表示根号，`\sqrt[n]{}` 表示 n 次根号。

| 含义 | 语法 | 效果 |
|:---|:---|:---|
| 平方根 | `\sqrt{x}` | $\sqrt{x}$ |
| 立方根 | `\sqrt[3]{x}` | $\sqrt[3]{x}$ |
| 复杂根号 | `\sqrt{b^2 - 4ac}` | $\sqrt{b^2 - 4ac}$ |

### 括号

普通括号大小固定，用 `\left(` 和 `\right)` 可以让括号自动适应内容高度。

| 含义 | 语法 | 效果 |
|:---|:---|:---|
| 小括号 | `\left( \frac{a}{b} \right)` | $\left( \frac{a}{b} \right)$ |
| 中括号 | `\left[ x \right]` | $\left[ x \right]$ |
| 绝对值 | `\left| x \right|` | $\left| x \right|$ |
| 花括号 | `\left\{ x \right\}` | $\left\{ x \right\}$ |

::: warning 注意
`\left` 和 `\right` 必须**成对出现**，否则会报错。
:::

## 常见问题

### 输入 $ 符号本身怎么办？

`$` 是公式标记符。如果需要在文本中显示美元符号 `$`，可以用转义 `\$`。

例如：`价格为 \$100` 会显示为：价格为 \$100

### 公式显示为红色或报错怎么办？

这通常是 LaTeX 语法错误。常见原因：

- 花括号不匹配：`\frac{a}{b}` 不能写成 `\frac{a}{b`
- 命令拼写错误：`\sqrt` 不能写成 `\squrt`
- `\left` 和 `\right` 没有成对出现

建议使用 **fx** 按钮插入模板，避免语法错误。

### 编辑器右侧预览不更新？

编辑器使用防抖机制（150ms），输入后稍等片刻预览会自动更新。如果仍不更新，尝试在编辑区点击一下或输入一个空格。

### 能否使用 Markdown 其他格式？

可以。编辑器同时支持标准 Markdown 语法：

- **加粗**：`**文字**`
- *斜体*：`*文字*`
- 列表：`- 项目`
- 标题：`## 标题`
- 代码：`` `代码` ``

公式语法可以与 Markdown 混合使用。
