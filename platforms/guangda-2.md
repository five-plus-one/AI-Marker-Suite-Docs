# 光大阅卷 V2

## 平台简介

光大阅卷 V2 是光大阅卷平台的另一种部署版本，通过 IP:端口方式访问，DOM 结构与主站（pj.yixx.cn）不同。

::: info 与光大阅卷的关系
光大阅卷有两个版本：
- **光大阅卷**（主站）：`pj.yixx.cn` — 标准域名访问
- **光大阅卷 V2**：IP:端口部署 — 内网或特殊部署环境

两个版本的 DOM 结构不同，使用不同的适配器。
:::

- **阅卷地址**: `http://{IP}:{端口}`（具体地址由部署方提供）

## 技术特点

- **前端框架**: Vue + Element UI
- **图片渲染**: Canvas（通过 XHR 拦截 API 获取图片 URL）
- **分数输入**: 点击式评分（`.scores li` 元素）
- **分数提交**: 点击"提交"按钮
- **图片获取**: XHR 拦截 API 响应，建立密号-图片映射池

## 使用方法

### 1. 安装脚本

安装 AI 批改助手最新版本。

### 2. 打开阅卷页面

通过 IP:端口访问光大阅卷 V2 平台，进入阅卷页面。

页面结构如下：

- **左侧**: 答题卡图片（通过 Canvas 渲染）
- **右侧**: 评分面板（点击式分数选项 + 提交按钮）

### 3. 使用 AI 批改

1. 脚本会自动检测阅卷页面，显示 AI 批改按钮
2. 点击按钮开始 AI 批改
3. AI 会自动识别答题卡图片、评分、点击对应分数选项并提交

## 图片获取

光大阅卷 V2 平台的答题卡通过 Canvas 渲染，脚本通过 XHR 拦截 API 响应获取图片 URL：

1. 拦截阅卷 API 响应，提取答题卡图片 URL
2. 建立密号（mh）与图片 URL 的映射池
3. 通过页面显示的密号精确匹配当前试卷图片

这种方式避免了图片预取导致的错位问题。

## 评分方式

光大阅卷 V2 使用**点击式评分**，通过 `.scores li` 元素选择分数：

- 分数选项：0, 1, 2, 3, ... N（根据满分值动态生成）
- 快捷按钮："满分"、"零分"
- 用户确认后才点击分数选项提交

脚本会自动点击与 AI 评分结果匹配的分数选项。
