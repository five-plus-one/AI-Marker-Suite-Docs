# 光大阅卷

## 平台简介

光大阅卷是一个在线阅卷平台，URL 格式为 `https://pj.yixx.cn/{考试ID}/#/painter`。

## 技术特点

- **前端框架**: Vue 2
- **图片渲染**: Canvas（多层 Canvas 叠加）
- **分数选择**: 点击式选择（0-5分），不是输入框
- **分数提交**: 点击"提交分数"按钮

## 使用方法

### 1. 安装脚本

安装 AI 批改助手最新版本。

### 2. 打开阅卷页面

登录光大阅卷平台，进入阅卷页面（URL 应包含 `/#/painter`）。

### 3. 使用 AI 批改

1. 脚本会自动检测阅卷页面，显示 AI 批改按钮
2. 点击按钮开始 AI 批改
3. AI 会自动识别答题卡图片、评分、填入分数并提交

## 评分模式

光大阅卷支持多种评分模式：

| 模式 | 说明 |
|------|------|
| 给分点 | 默认模式，点击预设分数选项 |
| 键盘给分 | 使用键盘输入分数 |
| 勾叉给分 | 使用勾/叉评分 |
| 减分模式 | 从满分开始扣分 |

AI 批改助手默认使用"给分点"模式，通过点击分数选项来填入分数。

## 小题支持

如果题目包含多个小题（如"2（4）"表示第2题满分4分），脚本会自动识别并支持分小题评分。

## 注意事项

### Canvas 跨域限制

光大阅卷使用 Canvas 渲染答题卡图片，部分 Canvas 可能因为跨域图片而无法直接导出。脚本会自动：

1. 尝试从可导出的 Canvas 提取图片
2. 如果 Canvas 导出失败，从网络请求中获取图片 URL

### 图片存储

答题卡图片存储在腾讯云 COS：
- 域名: `rescenter-1301246127.cos.ap-guangzhou.myqcloud.com`
- 路径格式: `/ptzx/markpic/dtk/{考试ID}/...`

## 故障排除

### 问题：脚本未显示 AI 按钮

**解决方案：**
1. 确认 URL 包含 `/#/painter`
2. 确认页面已完全加载（Canvas 和分数选项都已显示）
3. 刷新页面重试

### 问题：无法获取答题卡图片

**解决方案：**
1. 检查浏览器控制台是否有跨域错误
2. 尝试切换到"键盘给分"模式
3. 联系作者反馈问题

### 问题：分数未正确填入

**解决方案：**
1. 确认当前是"给分点"模式
2. 检查分数选项是否显示正确
3. 手动点击分数选项测试
