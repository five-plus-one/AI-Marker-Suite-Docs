# 科耘阅卷

## 平台简介

科耘阅卷是一个在线阅卷平台，使用 Vue + Element UI 框架，答题卡通过 SVG `<image>` 标签渲染。

- **阅卷地址**: `https://kaoshi.keewing.com`

## 技术特点

- **前端框架**: Vue + Element UI
- **图片渲染**: SVG `<image>` 标签（`xlink:href` 属性，带 OSS 裁切参数）
- **分数输入**: 输入框（`.score-input`）+ 满分/零分快捷按钮
- **分数提交**: 填入分数后平台自动提交（需勾选"自动提交"）
- **特殊功能**: 打分键盘、打分提示、痕迹工具栏

## 使用方法

### 1. 安装脚本

安装 AI 批改助手最新版本。

### 2. 打开阅卷页面

登录科耘阅卷平台，进入阅卷页面。页面 URL 的 hash 部分应包含 `/review/read/` 或 `/marking/`。

页面结构如下：

- **左侧**: 答题卡图片（通过 SVG `<image>` 渲染，带裁切参数）
- **右侧**: 打分栏（分数输入框 + 满分/零分按钮 + 提交按钮）

### 3. 使用 AI 批改

1. 脚本会自动检测阅卷页面，显示 AI 批改按钮
2. 点击按钮开始 AI 批改
3. AI 会自动识别答题卡图片、评分
4. 用户确认后，脚本填入分数并触发平台自动提交

::: warning 延迟填分模式
科耘平台在填入分数后会自动提交。脚本采用延迟填分模式：AI 识别分数后只记录不填入，等待用户确认后才真正填入分数，避免误提交。
:::

## 图片获取

科耘平台的答题卡通过 SVG `<image>` 标签渲染，图片托管在 OSS 上（kyasource.keewing.com），URL 中带有裁切参数：

```
http://kyasource.keewing.com/.../image.jpg?x-oss-process=image/crop,x_117,y_1079,w_663,h_210
```

裁切参数 `x_117,y_1079,w_663,h_210` 表示：起始坐标 (117, 1079)、宽度 663px、高度 210px。

脚本优先获取带裁切参数的图片 URL，只提取当前题目区域。

## 评分方式

科耘平台支持两种评分方式：

### 输入框评分

- 输入框：`.score-input`，placeholder 显示"满分N"
- 脚本通过模拟原生输入事件填入分数

### 快捷按钮

- "满分"按钮：一键填入满分
- "零分"按钮：一键填入零分

脚本会根据 AI 评分结果自动选择使用快捷按钮或输入框。

## 平台特殊处理

### 打分键盘

科耘平台的"打分键盘"功能会捕获所有键盘输入用于评分。脚本会在初始化时自动取消勾选"打分键盘"，避免用户输入数字时被平台误判为分数输入。

### 自动提交

脚本在填入分数前会自动勾选"自动提交"复选框，确保填分后平台能正常提交。
