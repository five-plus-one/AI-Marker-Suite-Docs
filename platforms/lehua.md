# 乐华阅卷

## 平台简介

乐华阅卷是一个在线阅卷平台，使用 Vue + Element UI 框架，答题卡通过 Canvas 渲染。

- **阅卷地址**: `https://main.lhsvr.cn`

## 技术特点

- **前端框架**: Vue + Element UI
- **图片渲染**: Canvas（通过 `canvas.toDataURL()` 导出）
- **分数输入**: 点击式评分按钮（`.score-btn`，0-N 分）
- **分数提交**: 点击"提交"按钮（Element UI 主按钮）
- **给分模式**: 点击按钮 / 输入框备用

## 使用方法

### 1. 安装脚本

安装 AI 批改助手最新版本。

### 2. 打开阅卷页面

登录乐华阅卷平台，进入阅卷页面。页面 URL 的 hash 部分应包含 `yuejuan` 或 `examYj`。

页面结构如下：

- **左侧**: 答题卡图片（通过 Canvas 渲染）
- **右侧**: 评分面板（包含 0-N 分的评分按钮、提交按钮）

### 3. 使用 AI 批改

1. 脚本会自动检测阅卷页面，显示 AI 批改按钮
2. 点击按钮开始 AI 批改
3. AI 会自动识别答题卡图片、评分、点击对应分数按钮并提交

## 图片获取

乐华阅卷平台的答题卡通过 `<canvas>` 元素渲染，脚本通过 `canvas.toDataURL()` 方法导出图片数据。

## 评分方式

乐华阅卷平台使用**点击式评分**，通过 `.score-btn` 按钮选择分数：

- 按钮组：0, 1, 2, 3, ... N（根据满分值动态生成）
- 按钮样式：Element UI 组件
- 备用方案：如果评分按钮不可用，支持输入框输入

脚本会自动点击与 AI 评分结果匹配的按钮。
