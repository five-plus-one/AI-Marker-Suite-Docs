# 智学网

智学网 (zhixue.com) 是 AI 批改助手最早支持的阅卷平台。

## 支持的页面

脚本在以下 URL 模式下自动激活：

- `*.zhixue.com/marking/*` — 阅卷页面
- `*.zhixue.com/webmarking/*` — Web 阅卷页面

## 功能支持

| 功能 | 支持状态 |
|------|---------|
| 自动获取答题卡图片 | 支持 |
| 自动填入分数 | 支持 |
| 自动提交 | 支持 |
| 等待下一份试卷 | 支持 |
| 分小题评分 | 支持 |
| 回评模式检测 | 支持 |
| 403 自动刷新 | 支持 |

## 分小题评分

智学网支持分小题评分。当页面上存在多个分数输入框时：

- 脚本自动识别 `#containter_topicTxt` 中的多个小题输入框
- 逐题填入分数
- 当仅检测到 1 个输入框时，自动跳过分小题模式

## 常见问题

### 页面检测失败

如果打开阅卷页面后没有出现 AI 批改按钮：

1. 确认当前页面 URL 匹配 `*.zhixue.com/marking/*` 或 `*.zhixue.com/webmarking/*`
2. 确认页面上显示了答题卡图片和分数输入框
3. 按 `F12` 查看控制台日志，检查是否有 `[诊断]` 信息

### 图片获取失败

智学网的答题卡图片可能有防盗链机制。脚本使用 `GM_xmlhttpRequest` 绕过跨域限制，正常情况下可以自动获取。

如果图片获取失败：

1. 检查网络连接
2. 刷新页面重试
3. 确认 Tampermonkey 权限设置正确

### 分数填入失败

如果 AI 评分正确但分数没有填入输入框：

1. 可能是页面结构更新导致选择器失效
2. 请更新脚本到最新版本
3. 如果仍然失败，请 [反馈问题](https://github.com/five-plus-one/AI-Marker-Suite/issues)

### 回评模式

在智学网的回评/复核模式下，脚本会自动隐藏 AI 批改按钮，避免误操作。
