# 粤教翔云智慧测评

## 平台简介

粤教翔云智慧测评是广东省教育厅推出的在线阅卷平台，使用 jQuery 传统页面架构。目前仅支持「题组批改」模式。

- **阅卷地址**: `https://rrtcp.gdedu.gov.cn`

::: warning 批改模式限制
粤教翔云平台有「题组批改」和「批量批改」两种模式。脚本仅支持「题组批改」模式，如果检测到「批量批改」模式会提示用户切换。
:::

## 技术特点

- **前端框架**: jQuery 传统页面
- **图片渲染**: `<img>` 标签
- **分数输入**: 输入框
- **分数提交**: 点击"提交"按钮
- **批改模式**: 仅支持「题组批改」

## 使用方法

### 1. 安装脚本

安装 AI 批改助手最新版本。

### 2. 打开阅卷页面

登录粤教翔云平台，进入阅卷页面。请确保使用「题组批改」模式。

页面结构如下：

- **左侧**: 答题卡图片
- **右侧**: 评分面板（分数输入框 + 提交按钮）

### 3. 使用 AI 批改

1. 脚本会自动检测阅卷页面，显示 AI 批改按钮
2. 点击按钮开始 AI 批改
3. AI 会自动识别答题卡图片、评分、填入分数并提交

## 图片获取

粤教翔云平台的答题卡通过 `<img>` 标签加载，脚本直接从 DOM 中获取图片 URL。

## 评分方式

粤教翔云平台使用标准的输入框评分方式，脚本通过模拟原生输入事件填入分数。
