# 鑫考

## 平台简介

鑫考网上阅卷是一个在线阅卷系统，通常部署在学校或教育机构的内部网络中，支持动态 IP 地址部署。

## 技术特点

- **前端框架**: jQuery 传统页面
- **图片渲染**: `<img class="teaimg">` 标签
- **分数输入**: 文本输入框（`#fenshu0`）
- **分数提交**: 点击"提交"按钮
- **平台检测**: 通过 URL 路径特征 + DOM 特征双重检测（兼容动态 IP）

## 使用方法

### 1. 安装脚本

安装 AI 批改助手最新版本。

### 2. 打开阅卷页面

登录鑫考平台，进入阅卷页面（URL 路径应包含 `/Marking/DisPlay`）。

### 3. 使用 AI 批改

1. 脚本会自动检测阅卷页面，显示 AI 批改按钮
2. 点击按钮开始 AI 批改
3. AI 会自动识别答题卡图片、评分、填入分数并提交

## 图片获取

鑫考平台的答题卡图片通过 `<img class="teaimg">` 标签渲染，脚本会自动从 `src` 属性获取图片 URL。

## 评分方式

鑫考平台使用**文本输入框**评分：

- 输入框 ID：`#fenshu0`
- 脚本会自动填入 AI 评分结果
- 支持分小题评分（多个输入框）

## 注意事项

### 动态 IP 部署

鑫考平台支持动态 IP 地址部署，脚本通过 URL 路径特征（`/Marking/DisPlay`）和 DOM 特征（`img.teaimg` + `#fenshu0`）双重检测，确保在不同部署环境下都能正确识别。

### 内网访问

鑫考平台通常部署在内网环境，脚本需要在能访问该平台的网络环境中使用。
