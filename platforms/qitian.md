# 七天网络

七天网络是 AI 批改助手支持的第二个阅卷平台，包含**旧版**和**新 UI** 两个适配器。

## 旧版 (7net.cc / qt7.net)

### 支持的页面

- `*.7net.cc/*` — 七天网络旧版
- `*.qt7.net/*` — 七天网络旧版备用域名

### 功能支持

| 功能 | 支持状态 |
|------|---------|
| 自动获取答题卡图片 | 支持 |
| 自动填入分数 | 支持 |
| 自动提交 | 支持 |
| 等待下一份试卷 | 支持 |
| 分小题评分 | 支持 |
| 回评模式检测 | 支持 |

## 新 UI (yj5.7net.cc)

七天网络新 UI 基于 Vue SPA 架构，答题卡使用 Canvas 渲染，适配难度较高。

### 支持的页面

- `yj5.7net.cc/*` — 七天网络新 UI

### 技术特点

新 UI 与旧版有显著差异：

| 特性 | 旧版 | 新 UI |
|------|------|-------|
| 框架 | 传统页面 | Vue SPA |
| 答题卡渲染 | `<img>` 标签 | Canvas |
| 图片获取 | 直接读取 src | API 拦截 |
| URL 变化 | 页面跳转 | 路由切换 |

### 图片获取策略

新 UI 的答题卡使用 Canvas 渲染，无法直接从 DOM 获取图片 URL。脚本采用以下策略：

1. **XMLHttpRequest 拦截** — 拦截 API 响应，精确获取当前学生的答题卡图片
2. **避免预取干扰** — 新 UI 会预加载下一份试卷的图片，脚本通过拦截确保只使用当前试卷的图片

### 功能支持

| 功能 | 支持状态 |
|------|---------|
| 自动获取答题卡图片 | 支持（API 拦截） |
| 自动填入分数 | 支持 |
| 自动提交 | 支持 |
| 等待下一份试卷 | 支持 |
| 分小题评分 | 支持 |
| 回评模式检测 | 支持 |

## 自动检测

脚本会自动检测当前页面属于哪个平台：

```
页面加载
    ↓
检查 URL 是否匹配智学网 → 加载智学网适配器
    ↓
检查 URL 是否匹配七天网络旧版 → 加载旧版适配器
    ↓
检查 URL 是否匹配七天网络新 UI → 加载新 UI 适配器
    ↓
都不匹配 → 脚本不激活
```

用户无需手动选择平台，脚本会在运行时自动判断。

## 常见问题

### 七天网络新 UI 图片获取失败

新 UI 使用 Canvas 渲染答题卡，如果 API 拦截失败：

1. 刷新页面重试
2. 检查脚本是否为最新版本
3. 确认页面上显示了答题卡

### 分小题填入失败

七天网络的分小题输入框结构可能因版本而异。如果自动检测失败：

1. 在设置面板中手动添加小题配置
2. 确认每个小题的标签和满分值正确

### 页面检测失败

如果脚本没有在七天网络页面上激活：

1. 确认 URL 匹配支持的域名
2. 确认页面上包含答题卡和分数输入框
3. 检查控制台日志
